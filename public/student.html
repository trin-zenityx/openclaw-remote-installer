OpenClaw Remote Installer

อาจารย์พร้อมช่วยติดตั้ง OpenClaw บนเครื่องของคุณแล้ว

เซิร์ฟเวอร์พร้อมใช้งาน
macOS / Linux
Windows

วิธีใช้งาน (macOS / Linux)

  1. เปิด Terminal
    macOS: กด Cmd+Space พิมพ์ "Terminal" แล้วกด Enter
    Linux: กด Ctrl+Alt+T
  2. กดปุ่ม คัดลอก ด้านบน
  3. วางคำสั่งใน Terminal (Cmd+V หรือ Ctrl+Shift+V)
  4. กด Enter เพื่อรัน
  5. เปิดหน้าต่าง Terminal ไว้ อย่าปิด

วิธีใช้งาน (Windows)

  1. เปิด PowerShell
    กด Win+X แล้วเลือก "Windows PowerShell"
    หรือ: กด Win พิมพ์ "PowerShell" แล้วกด Enter
  2. กดปุ่ม คัดลอก ด้านบน
  3. คลิกขวาใน PowerShell เพื่อวาง
  4. กด Enter เพื่อรัน
  5. ถ้าถูกถามเรื่อง execution policy ให้พิมพ์ Y แล้วกด Enter
  6. เปิดหน้าต่าง PowerShell ไว้ อย่าปิด
